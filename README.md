# DSValues

A website that allows players to check the value and demand of items in [Duel Stars](https://www.roblox.com/games/82983707799188/Duel-Stars).

## Purpose
- Quick item value/demand lookup
- Individual item pages via api routes
- Responsive and fast design

## Tech Stack
- Next.js
- React
- TailwindCSS

## Project Structure
- /app - pages and routes
- /components - reusable ui components
- /public - images and static assets
- /styles - global styles

## Notes
- node_modules, .next, and .env files are ignored via .gitignore.
- Keep commits small and descriptive.
- Do not give up no matter what.
